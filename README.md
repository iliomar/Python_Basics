## Propósito del repositorio

Este material fue creado para el **Taller de Python para Educadores**, ofrecido en la **Universidad de Puerto Rico en Mayagüez (UPRM)**.  
Está orientado a **maestros**, para que aprendan Python desde cero y lo integren en su enseñanza.

> El enfoque no es memorizar código, sino **comprender la lógica** y cómo aplicarla para enseñar ciencia.

## Estructura del taller

El contenido está dividido en **6 capítulos** más una introducción breve sobre Python y las herramientas usadas (Google Colab, Anaconda, etc.).

| Capítulo | Tema | Descripción breve |
|-----------|------|------------------|
| **1. `print()`** | Mostrar resultados | Cómo imprimir texto, cálculos o mensajes. |
| **2. Variables** | Guardar datos | Uso de variables, tipos (`int`, `float`, `str`, `bool`) y operaciones aritméticas. |
| **3. Condicionales** | Toma de decisiones | Uso de `if`, `elif`, `else` para clasificar o comparar valores. |
| **4. Bucles** | Repetición de acciones | Uso de `for` y `while` para ejecutar procesos iterativos. |
| **5. Listas** | Colecciones de datos | Cómo almacenar y recorrer múltiples valores. |
| **6. Funciones** | Reutilización de código | Crear y usar funciones para automatizar cálculos. |
